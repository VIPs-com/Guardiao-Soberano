✅ Nível Novato: Setup Híbrido Simples, Seguro e com Baixo Risco de Erro (2026)

Este é o ponto de entrada ideal para quem está começando na auto-custódia. O foco é simplicidade, segurança básica e processos que minimizam erros humanos. Comece com quantias pequenas (ex: R$ 500–5.000) para praticar e ganhar confiança antes de escalar.

Componentes Recomendados (Mais Usados pela Comunidade Iniciante)

  • Desktop (gerenciamento principal):
    Tails OS (live USB) + Sparrow Wallet ou Electrum.
    • Tails força Tor, é amnésico (apaga rastros ao desligar) e vem com Electrum pré-instalado.
    • Sparrow é o “canivete suíço” atual para Bitcoin (melhor que Electrum em coin control e privacidade para muitos).
  • Mobile (transações rápidas/daily):
    GrapheneOS ou CalyxOS em um Google Pixel (6a ou mais novo recomendado).
    • GrapheneOS prioriza segurança máxima (hardening).
    • CalyxOS é mais amigável para iniciantes (mais “pronto para usar”).
  • Hardware Wallet (cold storage – o mais importante):
    Trezor Safe 3 ou Trezor Safe 5 (mais fácil para novatos).
    • Alternativa: Coldcard Mk4/Q se quiser Bitcoin-only e air-gapped desde o início (um pouco mais complexo).
      Trezor é open-source, interface amigável e amplamente recomendado para beginners.

Passo a Passo Detalhado (Fluxo Seguro)

  1. Prepare o Desktop (Tails OS)
    • Baixe Tails do site oficial (tails.boum.org) e verifique a assinatura GPG.
    • Crie um USB bootável (use Rufus no Windows ou dd no Linux/Mac).
    • Boot no Tails → Configure Persistência Criptografada (Applications → Tails → Configure Persistent Volume).
      • Ative: Personal Documents, Dotfiles, e opcionalmente Network Connections.
      • Use uma senha forte para o volume persistente.
    • No volume persistente, baixe e instale Sparrow Wallet (versão Standalone Linux).
  2. Configure a Hardware Wallet
    • Compre direto do fabricante (trezor.io ou coldcard.com).
    • Inicialize o dispositivo offline.
    • Gere a seed BIP39 (24 palavras) → Anote em papel/metal (nunca digital).
    • Teste o restore com uma seed de teste pequena.
    • Crie uma passphrase extra (BIP39) para segurança adicional (guarde separada).
  3. Crie a Wallet no Sparrow/Electrum
    • No Tails + Sparrow:
      • Crie uma wallet watch-only (só endereços públicos) conectada ao seu hardware.
      • Ou use o modo hardware direto (USB ou QR no Coldcard).
    • Prefira Single Sig no início (mais simples). Depois migre para 2-of-3 multisig.
    • Conecte preferencialmente a um Electrum Server privado ou Bitcoin node (pode configurar depois).
  4. Mobile Setup
    • Instale GrapheneOS ou CalyxOS no Pixel (siga o guia oficial – leva ~30-60 min).
    • Apps recomendados:
      • BlueWallet ou Electrum (para Bitcoin).
      • Use watch-only (só vê saldo, não gasta sem hardware).
      • Orbot para rotear via Tor.
    • Mantenha só valores pequenos no mobile (hot wallet).
  5. Primeira Transação Segura (Teste)
    • Envie uma quantia pequena da exchange para o endereço gerado no Sparrow (watch-only).
    • Use PSBT (Partially Signed Bitcoin Transaction): assine offline com hardware.
    • Transmita a transação assinada do Tails.

Melhores Práticas de Segurança (Baixo Risco)

  • Nunca digite a seed em qualquer computador online.
  • Verifique sempre hashes e assinaturas GPG dos downloads.
  • Use o Tails para todas as operações sensíveis.
  • Backups: Seed em metal (ex: Billfodl ou placa de aço), guardada em locais diferentes. Teste o restore periodicamente.
  • Senhas fortes + passphrase na hardware wallet.
  • Comece pequeno e aumente gradualmente (conforme tabela Sparrow: até ~US$10k em single sig).
  • Evite Wi-Fi público; use Ethernet ou Tor no Tails.

Vantagens deste Setup Novato

  • Baixo custo inicial (~R$ 800–1.500 para Trezor + Pixel usado + USBs).
  • Tutoriais em português abundantes no canal Bitcoinheiros.
  • Fácil de evoluir para níveis intermediário (adicionar Qubes/Whonix, multisig, node próprio).
  • Forte proteção contra malware comum (Tails amnésico + hardware).

Tempo estimado para setup inicial: 4–8 horas + prática.
