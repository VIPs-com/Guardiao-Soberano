✅ Tutorial Detalhado: Instalação do Firmware Krux (Atualizado 2026)

Existem duas formas principais de instalar o firmware Krux:

  1. Mais fácil para novatos → Krux-Installer (GUI, recomendado).
  2. Via linha de comando (mais controlável, usa ktool).

Site oficial sempre consulte: selfcustody.github.io/krux

1. Preparação (Obrigatório)

  • Hardware: Maix Amigo, M5StickV, Yahboom, etc. (conectado via USB).
  • Computador: Tails, Linux, Windows ou macOS.
  • Baixe tudo do site oficial do Krux (GitHub releases).
  • Verifique sempre os arquivos (SHA256 + assinatura).

Método Recomendado: Krux-Installer (GUI – Mais Fácil)

  1. Acesse: github.com/selfcustody/krux-installer/releases
  2. Baixe a versão mais recente para seu sistema (Linux .AppImage, Windows .exe, etc.).
  3. Verifique o arquivo baixado (SHA256).
  4. Execute o instalador:
    • Linux: chmod +x krux-installer*.AppImage && ./krux-installer*.AppImage
    • Windows: Execute o .exe como administrador.
  5. No Krux-Installer:
    • Selecione seu dispositivo (ex: Maix Amigo, M5StickV).
    • Escolha a versão mais recente do firmware.
    • Clique em Download → o instalador verifica automaticamente a assinatura.
    • Conecte o dispositivo via USB (use a porta inferior no Maix Amigo).
    • Clique em Flash / Instalar.
    • Aguarde o processo terminar (pode reiniciar o dispositivo).

Recurso útil: Tem opção de Wipe Device (limpar completamente antes de instalar).

Método Alternativo: Via Linha de Comando (ktool)

  1. Baixe o release oficial do Krux:
    github.com/selfcustody/krux/releases
    (arquivo krux-vXX.XX.X.zip)

  2. Verifique os arquivos (muito importante):

    # No Terminal (Tails ou Linux)
sha256sum --check krux-*.zip.sha256.txt

    Verifique assinatura com selfcustody.pem (disponível no repo).

  3. Extraia o ZIP:

    unzip krux-v*.zip && cd krux-v*
  4. Conecte o dispositivo (ligado) via USB.
  5. Rode o comando de flash (substitua DEVICE):

    • Maix Amigo / M5StickV / Yahboom / Cube:

      ./ktool -B goE -b 1500000 maixpy_DEVICE/kboot.kfpkg

      (Ex: maixpy_amigo ou maixpy_m5stickv)

    • Para alguns modelos (dock, wonder_mv etc.):

      ./ktool -B dan -b 1500000 ...
  6. Aguarde o flash completar. O dispositivo deve reiniciar com o Krux.

No Tails: Use o Terminal no volume persistente. Pode precisar de permissões (sudo se necessário).

Após a Instalação

  1. Ligue o Krux → deve aparecer a tela inicial.
  2. Configure:
    • New Wallet → Gere seed com dice rolls (alta entropia).
    • Anote a seed em metal.
    • Ative TC Flash Hash (verificação de integridade do firmware).
  3. Teste exportação de xpub via QR (como explicado antes).
  4. Configure no Sparrow (Air Gapped → Krux).

Problemas Comuns e Soluções

  • Erro de porta/serial: Verifique com ls /dev/ttyUSB* (Linux). Use porta inferior no Amigo. Tente desconectar/reconectar.
  • Greeting fail / timeout: Use -B goE correto e baixe a velocidade se necessário.
  • Tela invertida no Amigo: Alguns modelos precisam de configuração extra (veja Troubleshooting oficial).
  • Dispositivo não detectado: Instale drivers USB (geralmente não precisa no Linux moderno).

Fontes oficiais: Documentação Krux (selfcustody.github.io/krux/getting-started/installing), GitHub releases e Krux-Installer.
