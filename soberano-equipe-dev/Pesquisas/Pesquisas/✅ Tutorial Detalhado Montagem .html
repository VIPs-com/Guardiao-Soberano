✅ Tutorial Detalhado: Montagem do SeedSigner (Atualizado 2026)

O SeedSigner é montado com componentes off-the-shelf baratos (geralmente < US$ 50-80). É um processo DIY simples, mas exige cuidado com os cabos e conexões.

1. Lista de Componentes Essenciais (oficial)

  • Raspberry Pi Zero v1.3 (sem WiFi/BT — preferencial para air-gap máximo). Versão W também funciona (com remoção de WiFi).
  • Waveshare 1.3" LCD HAT (240x240 pixels — confirme a resolução!).
  • Câmera compatível com Pi Zero (ex: AuviPal 5MP ou similar com cabo flat flex).
  • MicroSD Card (mínimo 64MB, classe 10 recomendada).
  • Enclosure (case 3D printed — várias opções open-source, como "Orange Pill").
  • Opcionais: GPIO Hammer Headers (solderless), cabo MicroUSB para alimentação.

Onde comprar (links oficiais no seedsigner.com/hardware/): Adafruit, Waveshare, AliExpress, The Pi Hut, etc. No Brasil, procure StackBit para cases ou componentes importados.

2. Preparação Antes da Montagem

  1. Flash da SD Card (faça isso antes de montar):
    • Baixe a imagem mais recente em: github.com/SeedSigner/seedsigner/releases.
    • Use Raspberry Pi Imager (ou dd no Linux/Tails) para gravar na SD.
    • Verifique o hash da imagem.
  2. Teste os componentes (recomendado):
    • Monte temporariamente sem o case.
    • Insira a SD, conecte alimentação → verifique se bootou e a tela/câmera funcionam.

3. Passos de Montagem (Fluxo Típico)

Baseado nos guias oficiais e vídeos de assembly (ex: Open Pill demo):

  1. Prepare o Raspberry Pi Zero:
    • Se os pinos GPIO não estiverem soldados, instale (solda ou use Hammer Headers solderless).
    • (Opcional para Pi Zero W): Remova/desabilite o módulo WiFi/BT para air-gap máximo.
  2. Instale a Câmera:
    • Conecte o cabo flat flex da câmera ao conector CSI do Pi Zero (contatos brilhantes virados para a placa verde).
    • Trave o cabo.
    • Remova a proteção da lente da câmera.
  3. Instale o Waveshare LCD HAT:
    • Alinhe o HAT com os pinos GPIO do Pi Zero.
    • Pressione firmemente para encaixar (não force demais).
    • O HAT tem joystick e botões integrados.
  4. Monte no Enclosure (Case):
    • Coloque o Pi Zero na base do case.
    • Posicione a câmera no slot correspondente (geralmente com clipes).
    • Encaixe o LCD HAT por cima.
    • Feche o case (muitos designs têm parafusos ou encaixe snap).
    • Exemplos populares: Open Pill (clássico laranja), Mini, ou cases mais robustos.
  5. Teste Final:
    • Insira a MicroSD.
    • Conecte alimentação MicroUSB.
    • O dispositivo deve bootar no menu do SeedSigner.
    • Teste câmera (scan QR), tela e joystick.

Vídeos recomendados para ver na prática (assista antes de montar):

  • Assembly Demo oficial: Open Pill Assembly
  • Tutorial completo: Southern Bitcoiner ou BTC Sessions no YouTube.

Dicas Importantes

  • Cuidado com cabos: O flat flex da câmera é delicado — não dobre em ângulos fechados.
  • Air-gap: Use Pi Zero 1.3. Nunca conecte rede.
  • Enclosure: Baixe STL no GitHub do SeedSigner e imprima em 3D (muitos makers no Brasil oferecem).
  • Kits prontos: Se preferir menos trabalho, vendedores como StackBit (Brasil), GoBrrr, etc., vendem kits ou montados (verifique reputação).

Após montado, siga para:

  • Gerar seed offline (dice rolls).
  • Exportar xpub via QR.
  • Integrar com Sparrow no Tails.
