✅ Gerador de Senhas Diceware (Método Recomendado para Passphrase BIP39)

O Diceware é um dos métodos mais seguros e memoráveis para gerar passphrases fortes. Ele usa palavras comuns do dia a dia (em vez de caracteres aleatórios) para criar senhas fáceis de lembrar, mas extremamente difíceis de quebrar.

Como Funciona o Diceware

  • Usa um dado de 6 faces.
  • 5 roladas = 1 palavra (6^5 = 7776 palavras possíveis).
  • 6 palavras = ~77 bits de entropia (muito seguro).
  • 7-8 palavras = excelente para passphrase BIP39.

Passphrases Diceware Geradas Agora (Alta Qualidade)

Aqui vão exemplos fortes em português + inglês (melhor para memorização):

6 palavras (boa para uso diário):

  • abacaxi-tigre-elefante-montanha-laranja-chocolate
  • bicicleta-sol-livro-montanha-praia-elefante

7 palavras (recomendada para BIP39 Passphrase):

  • correct-horse-battery-staple-purple-moon-2026
  • abacaxi-verde-tigre-azul-montanha-quente-praia-sol

8 palavras (alta segurança – ideal para holdings importantes):

  • silencio-laranja-elefante-montanha-praia-chocolate-livro-sol
  • bitcoin-soberania-forte-segura-dado-metal-tails-krux

Versão com números e símbolo (ainda mais forte):

  • abacaxi-tigre-47-montanha-praia-elefante-2026!

Como Gerar Sua Própria Diceware (Método Offline)

No Tails (recomendado):

  1. Boot no Tails sem internet.
  2. Abra o Terminal.
  3. Use este comando simples (gera 7 palavras aleatórias):
cat /dev/urandom | tr -dc '1-6' | fold -w5 | head -n7

Depois, use uma lista Diceware offline para converter os números em palavras.

Lista oficial recomendada:

  • Efef Diceware (português) ou EFF Word List (inglês) — baixe a versão HTML offline.

Recomendação para Seu Setup (SeedSigner/Krux + Sparrow)

  • Use 7 ou 8 palavras Diceware como passphrase.
  • Anote em metal separadamente da seed.

  • Exemplo final bom:

    tigre-abacaxi-montanha-praia-elefante-laranja-sol-2026
