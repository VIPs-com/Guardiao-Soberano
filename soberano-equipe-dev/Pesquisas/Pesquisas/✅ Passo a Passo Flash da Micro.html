✅ Passo a Passo: Flash da MicroSD Card para SeedSigner (Atualizado 2026)

Este é o processo mais seguro e recomendado. Faça isso antes de montar o hardware (ou pelo menos antes de fechar o case).

1. Prepare os Materiais

  • MicroSD Card de pelo menos 64MB (classe 10 ou melhor, 8GB+ recomendado).
  • Leitor de cartão SD no seu computador (ou adaptador USB).
  • Computador com internet (pode ser no Tails, mas Linux/Windows/Mac normal é mais fácil).

2. Baixe a Imagem Oficial

  1. Acesse o GitHub oficial:
    github.com/SeedSigner/seedsigner/releases
  2. Baixe a versão mais recente (ex: seedsigner_os.X.X.X.pi0.img).
    • Para Raspberry Pi Zero 1.3 (recomendado): use o arquivo pi0.img.
    • Verifique se há outros arquivos como .sha256 e .ascpara verificação.

3. Verifique a Integridade (Muito Importante!)

  1. Baixe também os arquivos de verificação (.sha256 e assinatura GPG).
  2. No Terminal (Linux/Tails) ou PowerShell (Windows):
# Verificar hash
sha256sum --check seedsigner_os.*.sha256.txt
  1. Verifique a assinatura GPG conforme instruções no README do release (chave do SeedSigner).

Se der OK, prossiga.

4. Flash da SD Card (Métodos Recomendados)

Opção A: Balena Etcher (Mais Fácil para a Maioria)

  1. Baixe e instale o Balena Etcher: balena.io/etcher
  2. Abra o Etcher.
  3. Clique em Flash from file → selecione a imagem .img baixada.
  4. Clique em Select target → escolha sua MicroSD Card (tenha cuidado para não escolher o HD do computador!).
  5. Clique em Flash! e aguarde (pode demorar 5-15 minutos).
  6. Quando terminar, ele vai verificar automaticamente.

Opção B: Raspberry Pi Imager (Muito Bom no Windows/Linux)

  1. Baixe o Raspberry Pi Imager: raspberrypi.com/software
  2. Abra o programa.
  3. Clique em Choose OS → Use custom → selecione o arquivo .img do SeedSigner.
  4. Clique em Choose Storage → selecione a MicroSD.
  5. Clique em Write e confirme.
  6. Aguarde o processo terminar.

5. Finalize e Teste

  1. Quando o flash terminar, ejete a SD Card com segurança.
  2. Insira a MicroSD no SeedSigner montado.
  3. Conecte a alimentação MicroUSB.
  4. Aguarde ~30-60 segundos: deve aparecer o logo do SeedSigner.
  5. Navegue com o joystick para confirmar que está funcionando.

Dicas de Segurança e Problemas Comuns

  • Backup da SD: Faça uma cópia da imagem antes de usar.
  • SD não é reconhecida? Formate a card como FAT32 primeiro (use o próprio Etcher ou SD Card Formatter).
  • Erro no flash? Tente outra porta USB, outra SD Card, ou mude de Etcher para Pi Imager.
  • Tails OS: Funciona, mas use o Terminal com dd se preferir (cuidado com o device correto: lsblk para identificar).
  • Atualizações futuras: Basta flashear uma nova imagem na mesma SD.

Após o flash, você pode prosseguir com a montagem final e geração de seed.
