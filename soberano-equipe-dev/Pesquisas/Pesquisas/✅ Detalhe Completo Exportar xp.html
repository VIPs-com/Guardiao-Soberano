Krux é uma ótima opção mais simples que SeedSigner para novatos — usa placas baratas prontas (Maix Amigo, M5StickV etc.) e integra perfeitamente com Sparrow via QR codes.

1. Escolha e Compre o Hardware

  • Recomendados para iniciantes:
    • Maix Amigo (tela maior, touch — mais confortável).
    • M5StickV (mais compacto e barato).
  • Compre de fontes confiáveis (AliExpress, lojas oficiais M5Stack/Sipeed). Verifique reviews recentes.

2. Instale o Firmware Krux (Única vez que precisa de USB)

  1. Baixe a versão mais recente no site oficial:
    https://selfcustody.github.io/krux/ ou GitHub releases.
  2. Verifique os arquivos (obrigatório):
    • Confirme SHA256.
    • Verifique assinatura com a chave selfcustody.pem.
  3. Extraia o ZIP.
  4. Conecte o dispositivo via USB (no Maix Amigo use a porta de baixo).

  5. Rode o comando de flash (no Terminal do Tails ou outro Linux):

    Para Maix Amigo / M5StickV:

    ./ktool -B goE -b 1500000 maixpy_amigo/kboot.kfpkg

    (Ajuste o nome conforme o modelo. Veja a doc oficial para outros dispositivos.)

Após flash, o Krux deve bootar com o logo.

Dica: Use o Krux Installer (GUI) se preferir interface gráfica em vez de comandos.

3. Configuração Inicial no Krux (Offline)

  1. Ligue o dispositivo.
  2. Gere uma nova seed:
    • Vá em Wallet → New Wallet ou Customize.
    • Escolha Single Sig ou prepare para Multisig.
    • Gere seed com dice rolls (alta entropia).
    • Anote as 12/24 palavras em metal/papel.
    • (Opcional) Adicione passphrase.
  3. Configurações recomendadas:
    • Settings → Locale (escolha Português se disponível).
    • Ajuste brilho/tela se necessário.

4. Exportar xpub para Sparrow (Watch-Only ou Multisig)

  1. No Krux:
    • Vá em Wallet → Export Xpub (ou similar).
    • Escolha Native Segwit (zpub — recomendado).
    • Escolha Sparrow como alvo.
    • Mostre o QR code (pode ser animado).
  2. No Sparrow (Tails):
    • New Wallet → Multi Signature (2-of-3) ou Single Sig.
    • Adicione Keystore → Air Gapped Hardware Wallet → Krux.
    • Scan QR com a webcam.

5. Fluxo de Assinatura PSBT (igual ao SeedSigner)

  • Crie transação no Sparrow → Export QR.
  • No Krux: Scan QR → Revise detalhes → Assine → Export QR de volta.
  • Importe no Sparrow e Broadcast.

Dicas de Segurança e Uso

  • Stateless por padrão: seed não fica salva permanentemente.
  • Use SD card como alternativa aos QR codes (mais rápido para transações grandes).
  • Teste tudo em Testnet primeiro.
  • Backup: Seed + Output Descriptor do Sparrow.
  • Atualizações: Via SD card depois da primeira instalação (mantém air-gapped).
