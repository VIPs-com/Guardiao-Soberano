Você construiu a fortaleza, mas todas as peças vieram da internet. Como saber que o arquivo que baixou não foi trocado por um impostor? Como garantir que o backup não foi corrompido? E como não gastar 10 minutos toda semana repetindo comandos?

1. Verificação PGP na Prática

Sempre que baixar um arquivo (Tails, Feather, Electrum, KeePassXC), o site oferece um arquivo .asc ou .sig (assinatura destacada) e a chave pública do desenvolvedor. O processo é:

Passo a passo (exemplo com Feather)

# 1. Baixe o AppImage e a assinatura
wget https://featherwallet.org/files/releases/linux/feather-2.7.0.AppImage
wget https://featherwallet.org/files/releases/linux/feather-2.7.0.AppImage.asc

# 2. Importe a chave pública do desenvolvedor (uma vez)
#    O fingerprint está no site oficial. Exemplo fictício:
gpg --keyserver hkps://keys.openpgp.org --recv-keys 0x1234567890ABCDEF

# 3. Verifique
gpg --verify feather-2.7.0.AppImage.asc feather-2.7.0.AppImage

Saída esperada: 

gpg: Assinatura feita em [data]
gpg: Assinatura correta de "Feather Dev <dev@featherwallet.org>" [confiança indefinida]
gpg: ATENÇÃO: Esta chave não é certificada por uma assinatura confiável!

A mensagem de "confiança indefinida" é normal — você não assinou a chave do desenvolvedor, apenas a importou. O importante é a linha "Assinatura correta". Se aparecer "Assinatura inválida", o arquivo foi adulterado.

E se a chave pública não estiver no keyserver?

Baixe o arquivo .asc da chave pública do site e importe:

gpg --import chave-publica.asc

2. Cifrando o Backup com GPG (entendendo cada pedaço)

O comando que você usou várias vezes merece um olhar mais profundo:

tar czf - \
  /home/amnesia/Persistent/feather-data \
  /home/amnesia/Persistent/electrum-data \
  ... \
| gpg --symmetric --cipher-algo AES256 --output /media/backup/fortaleza_backup_$(date +%Y%m%d).tar.gz.gpg
  • tar czf -: empacota e comprime (c=create, z=gzip, f=- envia para stdout).
  • | (pipe): conecta a saída do tar à entrada do GPG.
  • gpg --symmetric: cifra com senha simétrica (você digita a frase-senha).
  • --cipher-algo AES256: força o algoritmo AES-256.
  • --output: define o nome do arquivo final.
  • $(date +%Y%m%d): adiciona a data (ex: 20260712) ao nome.

Por que simétrico? Simples e direto. Uma única frase-senha memorizada (ou guardada no KeePassXC e em local separado). Não há chave privada para gerenciar, ideal para backups.

3. Script Mestre do Operador

Crie um script na persistência que automatize:

  1. Verificação de integridade de arquivos baixados.
  2. Backup 3-2-1 rápido.
  3. Restauração guiada.

O Script

Salve como /home/amnesia/Persistent/fortaleza-toolkit.sh:

#!/bin/bash

# ==============================================
# FORTALEZA TOOLKIT - Cypherpunk Operacional
# ==============================================

PASTA_APPS="/home/amnesia/Persistent/Apps"
PASTA_BACKUP="/media/backup"

# Cores para melhor visualização
VERDE='\033[0;32m'
VERMELHO='\033[0;31m'
AMARELO='\033[1;33m'
SEMCOR='\033[0m'

# Função: Verificar assinatura PGP
verificar_assinatura() {
    arquivo="$1"
    assinatura="$2"
    echo -e "${AMARELO}[VERIFICANDO]${SEMCOR} $arquivo"
    if gpg --verify "$assinatura" "$arquivo" 2>&1 | grep -q "Assinatura correta"; then
        echo -e "${VERDE}[OK]${SEMCOR} Assinatura válida!"
        return 0
    else
        echo -e "${VERMELHO}[FALHA]${SEMCOR} Assinatura INVÁLIDA ou erro!"
        return 1
    fi
}

# Função: Criar backup 3-2-1
criar_backup() {
    echo -e "${AMARELO}[BACKUP] Iniciando backup seguro...${SEMCOR}"
    echo "Digite a frase-senha GPG para criptografar o backup:"
    read -s FRASE

    ARQUIVO_SAIDA="$PASTA_BACKUP/fortaleza_backup_$(date +%Y%m%d_%H%M).tar.gz.gpg"

    tar czf - \
        /home/amnesia/Persistent/feather-data \
        /home/amnesia/Persistent/electrum-data \
        /home/amnesia/.local/share/haveno \
        /home/amnesia/Persistent/fortaleza_cofre.kdbx \
        /home/amnesia/Persistent/haveno_backup_amnesia.zip \
    | gpg --batch --passphrase "$FRASE" --symmetric --cipher-algo AES256 --output "$ARQUIVO_SAIDA"

    if [ $? -eq 0 ]; then
        echo -e "${VERDE}[OK]${SEMCOR} Backup salvo em: $ARQUIVO_SAIDA"
        # Opcional: copiar para um pendrive extra
        # cp "$ARQUIVO_SAIDA" /caminho/para/segundo_pendrive/
    else
        echo -e "${VERMELHO}[ERRO]${SEMCOR} Falha ao criar backup."
    fi
}

# Função: Restaurar backup
restaurar_backup() {
    echo "Backups disponíveis:"
    ls $PASTA_BACKUP/fortaleza_backup_*.tar.gz.gpg
    echo ""
    echo "Digite o nome exato do arquivo de backup:"
    read ARQUIVO
    echo "Digite a frase-senha GPG:"
    read -s FRASE

    gpg --batch --passphrase "$FRASE" --decrypt "$PASTA_BACKUP/$ARQUIVO" | tar xzvf - -C /

    if [ $? -eq 0 ]; then
        echo -e "${VERDE}[OK]${SEMCOR} Backup restaurado com sucesso."
    else
        echo -e "${VERMELHO}[ERRO]${SEMCOR} Falha na restauração."
    fi
}

# Menu principal
echo "========================================="
echo "       FORTALEZA TOOLKIT v1.0"
echo "========================================="
echo "1) Verificar assinatura PGP de um arquivo"
echo "2) Criar backup 3-2-1"
echo "3) Restaurar backup"
echo "4) Sair"
read -p "Escolha: " OPCAO

case $OPCAO in
    1)
        read -p "Caminho do arquivo (.AppImage, .iso): " ARQ
        read -p "Caminho da assinatura (.asc, .sig): " ASS
        verificar_assinatura "$ARQ" "$ASS"
        ;;
    2)
        criar_backup
        ;;
    3)
        restaurar_backup
        ;;
    *)
        echo "Até logo, cypherpunk."
        ;;
esac

Torne-o executável: 

chmod +x /home/amnesia/Persistent/fortaleza-toolkit.sh

4. Automatizando a Verificação de Todos os Downloads

Para quem quer um script que baixa e já verifica a assinatura dos aplicativos principais, crie este complemento:

#!/bin/bash
# Script: atualizar-fortaleza.sh

PASTA_APPS="/home/amnesia/Persistent/Apps"
cd $PASTA_APPS

# Feather
echo "Baixando Feather..."
torsocks wget -q https://featherwallet.org/files/releases/linux/feather-2.7.0.AppImage
torsocks wget -q https://featherwallet.org/files/releases/linux/feather-2.7.0.AppImage.asc
gpg --verify feather-2.7.0.AppImage.asc feather-2.7.0.AppImage

# Electrum
echo "Baixando Electrum..."
torsocks wget -q https://download.electrum.org/4.5.5/electrum-4.5.5-x86_64.AppImage
torsocks wget -q https://download.electrum.org/4.5.5/electrum-4.5.5-x86_64.AppImage.asc
gpg --verify electrum-4.5.5-x86_64.AppImage.asc electrum-4.5.5-x86_64.AppImage

# KeePassXC
echo "Baixando KeePassXC..."
torsocks wget -q https://github.com/keepassxreboot/keepassxc/releases/latest/download/KeePassXC-2.7.9-x86_64.AppImage
torsocks wget -q https://github.com/keepassxreboot/keepassxc/releases/latest/download/KeePassXC-2.7.9-x86_64.AppImage.asc
gpg --verify KeePassXC-2.7.9-x86_64.AppImage.asc KeePassXC-2.7.9-x86_64.AppImage

echo "Verificação concluída. Se não houver 'Assinatura incorreta' acima, tudo OK."

Se preferir fazer no olho:

  • Tails: O site fornece um checksum SHA256. Rode sha256sum tails.img e compare com o site.
