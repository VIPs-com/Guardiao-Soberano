🖨️ FASE 2 — VERSÃO IMPRESSA (GRÁFICA)

Especificações Técnicas para Gráfica

Formato do Livro

Arquivo para Gráfica

O arquivo que a gráfica precisa é diferente do PDF digital:

PDF DIGITAL (para leitura em tela):
- Tamanho A5
- Links clicáveis
- RGB
- Fonte de qualquer tamanho

PDF PARA GRÁFICA (PDF/X-1a):
- Tamanho A5 + 3mm de sangria (bleed) = 154 × 216mm
- SEM links (não funcionam no papel)
- CMYK (ciano, magenta, amarelo, preto)
- Imagens ≥ 300 DPI
- Fontes embedadas
- Marcas de corte (crop marks)

Software para Preparar o PDF de Gráfica

Passo a Passo para a Gráfica

1. Gerar o conteúdo final em Markdown
2. Converter para PDF via Pandoc (com template LaTeX)
3. Ou diagramar no Scribus/Affinity
4. Configurar:
   - Tamanho da página: 154 × 216mm (A5 + bleed)
   - Margens internas: 20mm (para lombada)
   - Margens externas: 15mm
   - Margens superior/inferior: 18mm
5. Exportar como PDF/X-1a
6. Conferir:
   - Todas as fontes embedadas
   - Imagens ≥ 300 DPI
   - Sangria correta
   - Número de páginas múltiplo de 4 (para cadernos)
7. Enviar para a gráfica com especificações
